#  Global Life Expectancy Analysis using Machine Learning & Power BI

## Overview

This project analyzes global life expectancy data and builds machine learning models to predict life expectancy based on economic, health, and social factors.

## Features

- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Machine Learning Models
- Model Comparison & Selection
- Power BI Dashboard Visualization

## Files

- `Life_Expectancy_Analysis.ipynb` – Python notebook
- `Data/` – Dataset files
- `Life_Expectancy_Dashboard.pbix` – Power BI dashboard
- `Life_Expectancy_Report.pdf` – Project report
- `Life_Expectancy_Presentation.pptx` – Project presentation

## Key Insights

- Higher GDP is associated with higher life expectancy
- Education has a positive impact on lifespan
- HIV/AIDS negatively affects life expectancy
- Vaccination rates contribute to better health outcomes
- Developed countries have higher life expectancy than developing countries

## Models Used

- Linear Regression
- KNN Regressor
- Decision Tree Regressor
- SVR
- Random Forest Regressor (**Best Model – R² ≈ 0.95**)

## Tools Used

- Python (Pandas, NumPy, Scikit-learn)
- Jupyter Notebook
- Power BI

## Objective

To analyze factors affecting life expectancy and predict life expectancy using machine learning and data visualization techniques.
